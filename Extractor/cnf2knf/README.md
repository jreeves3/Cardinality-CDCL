# CNF to KNF Extraction

## Build 

To build use

```bash
sh build.sh
```

To clean use

```bash
sh clean.sh
```

## General Usage

```bash
./cnf2knf test/maxsquare-7-33-UNSAT-total.cnf
```

Note, only ALK is enabled currently, with logging turned on. Command line options will be added in the near future.
